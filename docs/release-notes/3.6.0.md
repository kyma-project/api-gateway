## New Features
- Introduced a new `ExternalGateway` custom resource. The ExternalGateway controller automatically creates and manages Istio Gateway resources, EnvoyFilters for certificate validation and XFCC sanitization, and handles mTLS configuration for secure communication with external gateways. See [issue #2560](https://github.com/kyma-project/api-gateway/issues/2560).
- APIRule v2 now supports referencing ExternalGateway resources via the `externalGateway` field (format: `namespace/name`), enabling routing of external traffic. The field is mutually exclusive with the standard `gateway` field.
